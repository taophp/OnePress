Database

OnePress stores its data in a Postgreql database, which is much more powerfull than others open source relational databases. It handles it through models form Phalcon framework but in its original way, making it greeter. All objects handled by OnePress are childs of the Items class, have a UUID, Universally Unique Identifier (v4).
